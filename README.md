# KAU-FPO Platform

AI-Based Digital Platform for KAU-FPO Linkage Programme

## Overview

This platform is built to support the Kerala Agricultural University (KAU) and Farmer Producer Organizations (FPO) linkage program, providing a modern admin dashboard for managing agricultural data, operations, and communications.

## Features

- Built with Next.js 16, TypeScript, Tailwind CSS v4, and Shadcn UI
- Responsive and mobile-friendly design
- Customizable theme presets with light/dark modes
- Flexible layouts with collapsible sidebar
- Authentication and user management
- Dashboard modules for CRM and general operations

## Tech Stack

- **Framework**: Next.js 16 (App Router), TypeScript, Tailwind CSS v4
- **UI Components**: Shadcn UI
- **Validation**: Zod
- **Forms & State Management**: React Hook Form, Zustand
- **Tables & Data Handling**: TanStack Table
- **Tooling**: Biome, Husky

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, or pnpm

### Installation

1. Clone the repository

2. Navigate into the project directory:
   ```bash
   cd next-shadcn-admin-dashboard
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

Your app will be running at [http://localhost:3000](http://localhost:3000)
